# Scottie Siele Website

Responsive static website for Scottie Siele, featuring his BBQ work, barbering, personal-brand consulting, financial education content, sports analytics, and New York Giants fandom.

## Local preview

```bash
python3 -m http.server 8080
```

Open `http://localhost:8080`.

## Deployment

The project is ready for Vercel or GitHub Pages. No build command is required.

## Before public launch

Confirm Scottie's public contact method, current barber availability, business identity, and any professional credentials or regulated-service licenses. The site includes financial, wagering, and brand-persona disclosures.
